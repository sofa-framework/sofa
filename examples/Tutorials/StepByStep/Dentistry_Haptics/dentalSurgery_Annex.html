Mesh generation

Preliminary note:

Note that this scene requires CGAL plugin. You must compile it to use it. If you don't know how to compile it, visit http://wiki.sofa-framework.org/wiki/Using_CGAL.

Description:

This scene is not a simulation, it only generates a volumetric mesh from a surfacic mesh. Unlike surfacic meshes, volumetric meshes represent the interior volume of an object. The goal is to create this mesh and export it as a VTK file to use it later in another simulation.

Key points:
  • Loading of the surfacic mesh with the Mesh component.
  • The MeshGenerationFromPolyhedron component (only available with CGAL plugin) generates the interior volume geometry from the surfacic mesh.
  • We build a mesh from the generated geometry in a second Mesh component.
  • We export the mesh as a VTK file with the VTKExporter component.
Note:

The MeshGenerationFromPolyhedron component needs some crucial parameters to generate good quality meshes. See CGAL documentation for more information about this parameters (http://www.cgal.org/Manual/3.6/doc_html/cgal_manual/Mesh_3/Chapter_main.html).
