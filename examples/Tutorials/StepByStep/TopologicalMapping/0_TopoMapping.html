Topological Mapping (1/7)




Description

The objective of this tutorial is to create a simulation with topological mapping between a volume and a surface mesh. We will use some of the most common Sofa components, trying not to enter into all the details related to the template selection. This will be done in another tutorial.

Moreover, details about topological components can be found in topological tutorials.

Key points

The starting point is a scene with all the default components needed to handle collisions in a scene. For more explanation, see the step by step tutorial about scene creation.

Create a new scene in the Modeler. Delete the TreeCollisionGroupManager.
