Topological Mapping (3/7)


Description

In this step we add a mechanical behavior to the tetrahedral mesh using ODE solver, finite element and mass. Thus, the cylinder can be deformed by the gravity or the user.
Key points
  • EulerImplicitSolver component is used to solve the mechanical ODE system.
  • CGLinearSolver component is used to solve the linare system at each step of the ODE system.
  • TetrahedralCorotationalFEM is used to create tetrahedral finite element on this mechanical system.
  • DiagonalMass discribe a global mass of the mesh elemets.
  • Constraint components are used here to fix on extremity face of the cylinder.

Results

Adding these components create a mechanical behavior for this cylinder. Thus, adding some gravity on the scene, we can see that the cylinder start to bend. On this step, as there is no surface for visual model, we can see the finite elements.
